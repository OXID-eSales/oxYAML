Installation:

1. Alles aus dem Ordner copy_this an den entsprechenden Ort kopieren.
2. Im Adminbereich anmelden und unter Tools die install.sql ausführen.
3. Theme unter "Erweiterungen => Themes" aktivieren.
4. Aus dem Adminbereich abmelden
5. /tmp Ordner leeren.
6. fertig
